## Valko-UI Docs

### Routing

- Updated documentation routing to use Nuxt4's group folder syntax (parentheses), enabling flat URLs (e.g., `/docs/divider` instead of `/docs/layout/divider`).

### Menu Navigation

- The menu now automatically scrolls to the active item on page reload.
- Scrolls to the active item when navigating between pages.
- Scrolls to the active item when opening the mobile menu.
